# Scriptonizer

Утилита для быстрой вставки шаблонов ответов в чатах поддержки.

## Версии
- **personal** — расширенная версия для личного использования (прозрачность, мини-окно)
- **office** — безопасная версия для корпоративной среды (без интернета, без скрытых функций)

## Безопасность (office-версия)
- Не требует интернета
- Не собирает данные
- Не требует прав администратора
- Хранит данные локально в JSON
- Код доступен для аудита

## Сборка
```bash
pyinstaller --onefile --noconsole --name "Scriptonizer" scriptonizer_safe.py
