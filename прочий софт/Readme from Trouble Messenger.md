# 🐞 Trouble Messenger 4.0 — Ушастик

Финальная стабильная версия программы для сбора и отправки отчётов о проблемах на ПК.

## 📦 Возможности

- Быстрый отчёт (скрин + описание + основные данные системы)
- Расширенный отчёт (дополнительно: сеть, видеокарта, диски, топ процессов)
- Отправка по почте (любому получателю, с копией отправителю)
- Тихий режим (окно сворачивается перед скриншотом)
- Папка с отчётами (все сохранённые файлы в одном месте)
- Логирование действий (только локально, в письма не отправляется)

## ⚙️ Как пользоваться

1. Опишите проблему
2. Заполните поля почты (отправитель + пароль приложения)
3. Укажите получателя (любой email)
4. Нажмите одну из кнопок:
   - **Быстрый отчёт** — сохранит HTML‑файл на диск
   - **Расширенный отчёт** — сохранит HTML‑файл с полной диагностикой
   - **Отправить по почте** — отправит скриншот и описание на указанный email

## 🔐 Требования к почте

- Для **Gmail, Yandex, Mail.ru** требуется **пароль приложения** (создаётся в настройках безопасности)
- Для **Outlook** можно использовать обычный пароль от учётной записи
- Поле **«Кому отправить»** принимает **любой email** (разработчик, мастер, знакомый)

## 📁 Структура

После запуска программа создаёт папки:
- `Reports` — сохранённые отчёты (HTML)
- `Logs` — локальные логи (для отладки)

## 🖥️ Системные требования

- Windows 7 / 8 / 10 / 11
- Не требует установки дополнительных библиотек (всё встроено в `.exe`)

## 📥 Установка

Скоро появится финальный релиз `TM_4.0.exe`

## 📄 Лицензия

Свободное использование для частных лиц. Коммерческое использование запрещено. Авторство сохраняется.
