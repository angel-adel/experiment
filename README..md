# Trouble Messenger 

## Как пользоваться:
1. Запустить TroubleMessenger.exe
2. Ввести технический email и пароль приложения
3. Указать email разработчиков (можно несколько через запятую)
4. Ввести свой email для копии
5. Описать проблему
6. Нажать кнопку — окно свернётся, сделает скрин и отправит разработчикам ошибку так же копией  на любую почту сотрудника, которая будет укащана, как резервная
